# Dashboard Features - Complete Overview

## 1. CHARTERER DASHBOARD

- Featured cargo display (highest profit, low risk) with AI analysis
- Other cargoes list for comparison
- Quick stats (total analyzed, ready for decision, average profit/confidence)
- AI analysis with agent status and confidence scores
- Recent activity feed with timeline of decisions
- Refresh analysis to re-trigger AI for all cargoes
- Compare multiple cargoes side-by-side
- Filters by profit range, risk level, bunker availability, status
- Personalized time-based greeting
- Sortable cargoes table (route, profit, risk, status, vessel)
- Detailed cargo view with bunker plan, AI reasoning, alternatives
- Bulk actions (Fix, Reject, Request Alternative)
- Search by route, vessel, or cargo ID
- Fixed cargoes tracking with status
- Fix cargo with confirmation
- Reject cargo with reason
- Request alternative bunker plans from AI
- View full cargo details
- Ask AI interactive questions about cargo
- Skip to next cargo
- Performance dashboard with key metrics (total profit, cargoes fixed, success rate, avg profit)
- 12-month profit trend chart (actual vs target)
- Bunker cost analysis (total spend, average price, market comparison, top ports)
- Fleet utilization metrics (active vessels, utilization rate, performance breakdown)
- Top performing routes ranked by profit
- AI performance insights and recommendations
- Time range filters (this month, last 30 days, quarter, YTD)
- Export performance report
- Email report to team

## 3. VESSEL DASHBOARD

- Daily brief (vessel name, voyage route, current position, ETA)
- Today's status (pending actions, report submission status, alerts count)
- Critical alerts with dismiss functionality
- ROB report reminder with quick submit
- Current voyage performance (speed, consumption, efficiency metrics)
- AI insights with performance recommendations and weather alerts
- Upcoming events timeline (bunkering, reports, weather changes)
- Quick actions (submit report, contact operator, view details, map, trends, settings)
- Vessel information (name, IMO, status, position, route)
- Validation checklist (ROB confirmed, consumption verified, plan feasible, crew briefed)
- Approve voyage plan
- Flag issues to operator
- Voyage plan details (route, distance, duration, progress, ETA next port)
- Bunker plan and ROB display (VLSFO/LSMGO with progress bars, estimated consumption, required quantity, next port schedule)
- ROB report form (date/time UTC, position lat/lon)
- Fuel tanks input (VLSFO tanks 1-3, LSMGO tank 4, total calculation)
- Consumption tracking (VLSFO and LSMGO last 24h)
- Operating mode selection (Laden/Ballast)
- Weather conditions (sea state, wind Beaufort scale)
- Comments field
- Upload tank photos
- Submit ROB report with draft auto-save
- Last ROB summary with trend analysis and consumption comparison
- Upcoming bunker schedule (port, ETA, quantity, supplier)
- Pre-bunkering checklist (plan received, supplier confirmed, berth arranged, crew briefed, tanks prepared, equipment checked, communication established)
- Mark all checklist items complete
- During bunkering form (barge/truck alongside time, start pumping time, sample checkboxes, flow rate, running total, temperature, issues/observations)
- Pause bunkering operation
- Complete bunkering operation
- Post-bunkering report (final sounding, quantity received, BDN number, quality check Pass/Fail, upload BDN, upload sample photos, final comments)
- Submit bunker report
- History timeline (chronological events)
- Search history functionality
- Filter by event type
- Event types (ROB reports, bunker operations, issues, approvals)
- Event details (timestamp, description, related data)
- Flag issues (ROB discrepancy, higher consumption, port accessibility, weather/route concerns, equipment issues)
- Issue description field
- Submit issue notification to operator
